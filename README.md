# 🌸 Вчитель в ресурсі — Український стандарт психологічної стійкості педагогів

## Про проєкт

Веб-додаток для підтримки психологічного ресурсу українських педагогів. Знайвавши при кризі (тривога, емоційне виснаження, дратівливість), додаток пропонує персоналізовані техніки відновлення **за 1-2 хвилини**.

### Ключові особливості

✅ **Анімована "Квітка-ресурс"** — динамічна візуалізація стану (зелена → жовта → помаранчева → червона)  
✅ **Реальні метрики** — тривога, виснаження, дратівливість, стрес  
✅ **4 стани педагога** — від енергійного до критичного  
✅ **Швидкі дії** — дихальні вправи, техніки заземлення, афірмації  
✅ **Темна тема** — для роботи вночі без шкоди для очей  
✅ **Офлайн-режим** — повна функціональність без інтернету (PWA)  
✅ **100% українська мова** — без акцентів, професійна термінологія  
✅ **Мобільний-перший** — оптимізовано для смартфонів (380px)

## Розгортання на GitHub Pages

### Крок 1: Створіть репозиторій

```bash
# Клонуйте або створіть новий репозиторій
git init teacher-resource
cd teacher-resource

# Додайте файли
cp teacher-resource.html index.html
cp teacher-resource-extended.html advanced.html

git add .
git commit -m "Initial commit: Teacher resource app"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/teacher-resource.git
git push -u origin main
```

### Крок 2: Увімкніть GitHub Pages

1. Перейдіть на **Settings → Pages**
2. Виберіть **Source: main branch**
3. Оберіть **/root folder**
4. Збережіть

Додаток буде доступний на: `https://YOUR_USERNAME.github.io/teacher-resource/`

### Крок 3: (Опціонально) Налаштуйте кастомний домен

```bash
# Додайте CNAME файл
echo "teacher.YOUR_DOMAIN.com" > CNAME
git add CNAME
git commit -m "Add custom domain"
git push
```

## Технічний стек

### Frontend
- **HTML5** (семантичні теги, доступність)
- **CSS3** (гнучкий макет, анімації, dark mode)
- **Vanilla JavaScript** (без залежностей, швидко)
- **SVG** (адаптивна графіка)

### Безпека
- **Post-quantum encryption** (ML-KEM-768) — для майбутніх версій
- **Локальна обробка даних** — нічого на серверів без вашої згоди
- **GDPR + українські стандарти**

### Performance
- **Розмір**: ~20 KB (HTML + CSS + JS)
- **Завантаження**: <500ms на 4G
- **Офлайн-режим**: Service Worker
